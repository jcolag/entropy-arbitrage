## Mini-Server, Part 20

While, once again, I don't know if I'll *play* it often, reviewing my notes drew my attention back to [Elevator Saga](https://play.elevatorsaga.com/), a JavaScript programming game.  I tried deploying it to my server, and it worked like a dream.  It has no build step or server, so I could drop the entire repository on a computer's default web directory and it works immediately.

The mini-server family may also expand soon.  I haven't had time to inspect much of it, but I found another pile of older computers.  What do I mean by pile?

- Another mini-PC, which *might* have preceded the former media server that doesn't work, or I may have forgotten about it.
- A netbook, which boots to [Lubuntu](https://lubuntu.me/) though I don't recall what password I used, and has a handful of dead keys on the keyboard.
- A Pinebook, missing the power adapter.
- Raspberry Pi-like single-board computers from old Kickstarter campaigns, two from [Udoo](https://www.udoo.org/) and two from [Zero ASIC](https://www.zeroasic.com/)'s Parallella project.

I doubt that the netbook will have any real use, though since it can tolerate a USB keyboard (unlike some old laptops), it might make a fun way to try out [Collapse OS](https://collapseos.org/).  If the mini-PC works, and doesn't collapse under load like the other one, then it'll become the new media server.  If I can scrounge up a power adapter for it (5 V/3 A, I believe), then the Pinebook will probably become the "bedroom laptop" (and spare) running scripts and the noise generator, while the current old laptop returns to its place as the media server.  And I haven't the foggiest idea if I can get any use out of the single-board computers, but maybe I'll come up with something.

It always seems like a complete waste to throw out computers that might have even the *tiniest* bit of use squeezed out of them, so even where I've yanked out hard drives to save the data, I have computers going back decades.  And then I forget about them, because I don't bother to keep track of that sort of thing.
